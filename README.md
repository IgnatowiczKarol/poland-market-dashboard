# Poland Market Dashboard

## Project overview

This project presents a static snapshot of the Polish financial market as of 02.06.2026.

The dashboard monitors:

* WIG20
* mWIG40
* sWIG80
* USD/PLN
* EUR/PLN
* Gold price in PLN

## Key metrics

* 1-day return
* 5-day return
* Month-to-date return
* Return since 06.03.2026
* 20-day annualized volatility
* Drawdown

## Tools and data sources

* Microsoft Excel
* Power Query
* GPW market data
* NBP API

## Project workflow

Raw data → Data cleaning → Return calculations → Dashboard → Market commentary
